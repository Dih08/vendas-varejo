# 📊 Análise de Vendas no Varejo

## 📌 Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de analisar um conjunto de dados contendo aproximadamente 120.000 transações de vendas no varejo.

A análise busca transformar dados brutos em informações estratégicas para apoiar a tomada de decisão empresarial, identificando padrões de vendas, sazonalidade, desempenho de categorias e comportamento regional dos consumidores.

## 🎯 Objetivos

- Analisar o faturamento ao longo do tempo.
- Identificar as categorias mais lucrativas.
- Comparar o comportamento de compra entre regiões.
- Gerar insights para estratégias comerciais e de marketing.

## 🛠 Tecnologias Utilizadas

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📂 Estrutura da Análise

### 1. Carregamento e Exploração dos Dados

Importação do dataset e preparação das informações para análise.

### 2. Faturamento Mensal

Visualização da evolução do faturamento ao longo do tempo para identificar tendências e sazonalidade.

### 3. Top 10 Categorias por Faturamento

Análise das categorias que mais contribuem para a receita da empresa.

### 4. Distribuição do Valor de Venda por Região

Comparação do comportamento de compra dos clientes em diferentes regiões utilizando boxplots.

## 📈 Principais Insights

### Sazonalidade das Vendas

Foi possível identificar períodos de maior e menor faturamento, permitindo melhor planejamento de estoque e campanhas promocionais.

### Concentração de Receita

As categorias líderes representam uma parcela significativa do faturamento total, indicando oportunidades de expansão e riscos de dependência.

### Diferenças Regionais

O ticket médio e a dispersão das vendas variam entre regiões, sugerindo a necessidade de estratégias comerciais específicas para cada mercado.
